WebWalletAPI

A digital wallet specification for the Web

The WebWallet API

Abstract

This specification defines an API that aims to allow Web applications to carry out electronic commerce transactions using WebWallets.

What is a WebWallet?

A WebWallet is a digital wallet that can temporarily store money on the Web for intended online purchases.
